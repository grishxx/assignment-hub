# assignment-hub
Overview:
Assignment Hub is a web application designed to simplify assignment submission and grading in an educational environment. It provides dedicated portals for students and teachers with an intuitive and responsive interface.

Features:
Student Login & Registration
Teacher Login & Registration
Assignment Submission (Individual & Team)
Assignment Grading
Public Grade & Ranking Dashboard
Excel Export of Grades
Dark & Light Theme
Local Storage for Data Persistence
Responsive UI
Animated User Interface

Technologies Used:
HTML5
React 18 (CDN)
JavaScript (ES6)
Tailwind CSS
Lucide Icons
SheetJS (Excel Export)
Local Storage API

Future Improvements:
Backend database integration
Cloud file storage
Email notifications
Role-based authentication
Admin dashboard
